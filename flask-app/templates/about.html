Sobre o detector de mentiras/ fake news




Esse detector de fake news foi elaborado através de um algoritmo de Machine Learning. Utiliza-se de um modelo NLP para classificar o texto como uma notícia potencialmente falsa, ou verdadeira. Este algoritmo foi treinado com o Fake.Br Corpus -- um dataset com 7.200 notícias, onde metade é falsa. Esse procedimento foi feito de maneira manual para garantir maior precisão possível.

A precisão do algoritmo do detector é de 92.84 %.

  • Accuracy: 92.85 %
  • Precision: 92.19 %
  • Recall: 93.36 %
  • Specificity: 92.35 %
  • Misclassification Rate: 7.08 %
